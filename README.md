CARA KOMPILASI MENGGUNAKAN MAKEFILE:
1. Buka Terminal atau Command Prompt.
2. Arahkan ke folder tempat file-file ini disimpan.
3. Ketik perintah berikut lalu tekan Enter:
   make

CARA MENJALANKAN PROGRAM:
- Pada OS Windows     : Ketik 'main' atau 'main.exe' lalu tekan Enter.
- Pada OS Mac/Linux   : Ketik './main' lalu tekan Enter.

CARA MEMBERSIHKAN FILE KOMPILASI (OPSIONAL):
- Jika Anda ingin menghapus file .o dan .exe yang terbuat secara otomatis saat kompilasi, jalankan perintah:
make clean
